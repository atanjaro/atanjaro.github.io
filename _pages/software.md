## Software

Here are some software projects I have been involved with:

### VariationalMC

- This package implements the Variational Monte Carlo (VMC) method for the Hubbard model and (to be added) models of electron-phonon interaction
- [GitHub](https://github.com/atanjaro/VariationalMC.jl)

### SmoQyDQMC
<picture>
  <source srcset="/images/smoqysuite.png" media="(prefers-color-scheme: dark)">
  <img src="/images/smoqysuite.png" alt="Holstein model figure" style="width:50%;">
</picture>
- This package implements the determinant quantum Monte Carlo (DQMC) method for Hubbard, and electron-phonon interactions, including both Holstein and Su-Schrieffer-Heeger (SSH) style electron-phonon coupling.
- [GitHub](https://github.com/SmoQySuite/SmoQyDQMC.jl)
